Java模拟面试 我表示很慌张

2020-07-25 评论 字数统计: 1.4k(字) 阅读时长: 5(分)

跟我聊聊JVM？
JVM功能相当于将Java程序翻译给操作系统，所以Java是跨平台的。
它整个体系分为栈本地方法栈（私有）堆程序计数器方法区（共有）。
举个例子，我们创建一个Student类，经过编译之后变成class文件。然后，经过类加载器变成class对象，方法区中存放所有类信息和静态变量，栈中存放引用对象变量等信息，堆中存放对象的实例。
这其中，为什么静态变量和类信息放到一起？因为在jdk1.8之前，堆根据垃圾回收的情况分成年轻代 老年代和永久代。永久代意味着不会被垃圾回收。然而，因为class对象是动态加载的，不好控制永久代的大小。所以，jdk1.8之后把这永久代这一块物理空间存放到非jvm内存中，叫做元空间。
垃圾收集算法有哪些？
复制算法：将堆年轻代分成Eden from to区，第一次垃圾回收后，幸存的都往to 区跑，to区变from区。对象存活率高的话，复制效率低下。标记-清除：第一次清 理，根据可达性分析将不可达对象标记，下次清理。产生大量内存碎片。
标记-清除-整理算法，在标记清除基础上，对对象进行整理，减少其内存碎片。
CMS基于标记清除算法
G1将所以垃圾分成大致Regin块，预计块中垃圾比较多的进行清理

Spring Aop Ioc?
Ioc控制翻转将创建对象的控制权交给Spring容器
具体先扫描包，在配置文件中读取类的全路径，通过反射创建class对象并加载到IOC容器中。
Aop主要是将辅助业务和主业务编织在一起，例如日志功能的实现。具体通过动态代理，对主业务方法的增强。可以使用注解的方式，对辅助性业务类进行实现，并指定要代理的对象路径。
SpringMVC流程？
通过dispatchServlet拦截所有请求，请求通过控制器层连接视图层和模型层，通过模型层提供的数据显示到视图上面。
MyBatis？
MyBatis简单实现了JDBC的框架，优秀的持久层框架。其中，MVC中的Dao层就是面向操作数据库的半自动化的ORM持久化层。JPA是自动化的ORM。
xml文件对sql语句灵活的操作。其中，返回类型可以由ResultMap进行构建， property表示java类属性名 column表示数据库列表名。
进行多对一关联的话要使用associate javaType表示关联的一。
一对多的话要用collection ofType因为返回多的是集合。
动态代理和静态代理有什么区别？
代理主要对方法的增强。实现就是委托类实现相同的接口。因此静态代理接口如果添加了方法，所有代理类也要添加，增大了维护的难度。为了解决它我们可以使用动态代理，用一个代理类就可以实现多个委托类的代理（创建一个代理机构）。newProxyInstance动态实现委托类相同接口的实现类，调用invoke（）方法进行转发实现调用方法的增强。（JDK动态代理）
单例模式？
饿汉和懒汉。饿汉是刚开始加载的时候就创建。懒汉是调用的时候进行创建。
其中，懒汉是不安全的。

线程不安全。双重检测机制 防止ab都进入判断前构建两次。

WebSocket 前端主动监听服务端消息，实时的更新数据。
Http协议和Https协议？
服务端通过向第三方平台获取证书，构造公钥发送给客户端，客户端接收公钥对私钥进行加密。发送给服务端，服务端利用公钥进行解密。保证了安全。
Shiro执行工作原理？
Shiro主要有两大功能，认证和授权
前端先访问，Shiro认证通过之后返回一个SessionId。下次前端再进行访问的时候，头信息带有sessionId，根据sessionId返回授权的信息。
后端，Shiro构造安全数据进行认证，主要通过MD5和盐保证数据安全性，再通过Shiro的session将信息存入redis中。下一次登录时，先看有没有sessionId,有的话根据sessionId获取登录的用户信息。
Nginx
负载均衡，可以通过部署两个Tomcat在Nginx上进行反向代理，其中，Nginx处理静态资源相比于Tomcat会比较快。但是如何保证其Session一致性呢？
定时器怎么理解？

  • 触发器设置时间
  • 工作内容写在Job中
  • 找到调度器进行对工作进行调度
    什么是HashMap？
    键值对的形式存在，查找数据比较快的一种数据结构。底层是数组加（链表和红黑树）
    快速排序和归并排序。
